History

  1. 第一个阶段:1949年以前
    1. 古典加密
    2. 1883年Kerchoffs第一次明确提出了编码的原则：加密算法应建立在算法的公开不影响明文和密钥的安全。

  2. 第二个阶段:1949年到1976年

    1. 标志: Shannon 发表”Communication Theory of Secrecy System”
    2. 密码学进入了科学的轨道
    3. 主要技术: 单密钥的对称密钥加密算法
    4. 1967年David Kahn的《The Codebreakers》许多人了解了密码学
    5. 1971-73年IBM Watson实验室的Horst Feistel等几篇技术报告
      • 主要特点：数据的安全基于密钥而不是算法的保密

  3. 第三个阶段 :1976年以后

    1. 标志 : Diffie,Hellman发表”New Directions in Cryptography”
    2. 一种新的密码体制 : 公开密钥体制
    3. Rivest,Shamir & Adleman RSA算法

Symmetric cipher Model

Cryptosystem

  • Quintuple (E, D, M, K, C)
  • M set of plaintexts
  • K set of keys
  • C set of ciphertexts
  • E set of encryption functions e: M * K -> C
  • D set of decryption functions d: C * K -> M

Language Redundancy and Cryptanalysis

  • in English E is by far the most common letter
  • followed by T,R,N,I,O,A,S
  • other letters like Z,J,K,Q,X are fairly rare

Block vs Stream Ciphers

  • block ciphers process messages in blocks, each of which is then en/decrypted
    • like a substitution on very big characters
    • 64-bits or more
  • stream ciphers process messages a bit or byte at a time when en/decrypting
    • many current ciphers are block ciphers
    • broader range of applications
    • stream ciphers RC4, A5/1, A5/2, A5/3
