  2. 将方程组改写为X = BX

实现

每个未知数分配给一个进程。 对进城 i而言：

x[i] = b[i];                                          // 初始化  
for (iteration = 0; iteration < limit; iteration++) {
    sum = -a[i][i] * x[i];
    for (j = 0; j < n; j++)
        sum = sum + a[i][j] * x[j];
    new_x[i] = (b[i] - sum) / a[i][i];    //新的临时值   broadcast_recv(&new_x[i]);        /*bcast/rec values */
    global_barrier();                          /* wait for all procs */
}
